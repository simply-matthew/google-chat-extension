Read Receipts

Off

Google Chat will mark messages as read normally.

Works on chat.google.com and Chat inside Gmail. Blocks the specific network calls Chat uses to mark a conversation read. Because that endpoint is shared with a few other Chat actions, some functionality may not work while this is on — flip it off again once you're done reading.

Hotkey Delete

Off

Deletes your most recent message instantly — no undo.

Press Ctrl+Shift+Backspace (⌘+Shift+Backspace on Mac) to instantly delete your last sent message in the open chat — no confirmation prompt. Deletes only your own messages.

Message History

Off

See original text when a message is edited or deleted.

While on, an ✏️ (edited) or 🗑️ (deleted) icon appears next to any message that changes — click it to see the original text. Only logs messages you've actually seen render going forward; stored locally on this device only.
